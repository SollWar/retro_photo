# Retro Photo

Статическое Next.js приложение, которое использует камеру устройства в браузере, позволяет переключать объективы, применять ретро-фильтры и сохранять готовые JPG-снимки на устройство.

## Запуск

```bash
bun install
bun run dev
```

## Статическая сборка

```bash
bun run build
```

После сборки готовый статический сайт лежит в `out/`.

## GitHub Actions

В проекте уже добавлены workflow для:

- автоматической сборки на каждый `push` и `pull request`
- автоматического деплоя `out/` на GitHub Pages при пуше в `main` или `master`

Для публикации на GitHub Pages нужно в репозитории открыть `Settings -> Pages` и выбрать `Source: GitHub Actions`.

## Возможности

- работа только на фронтенде, без сервера
- доступ к `MediaDevices` и переключение между камерами
- мощные ретро-фильтры с сохранением эффекта в итоговом файле
- зерно, виньетка и дата на кадре в настройках
- экспорт снимков в JPG прямо на устройство
